// What it does

Everything you need.
Nothing you don't.

Ghost Mode is built around one idea — personal protection, not complexity.

  • 🎙️

    MIC / CAMERA MONITOR

    Catches any process secretly accessing your microphone or camera hardware and logs it immediately.

  • 🌐

    NETWORK MONITOR

    Watches all outbound connections in real time. Flags unknown IPs and ports commonly used by spyware.

  • 👻

    ARP SPOOF DETECTOR

    Detects man-in-the-middle attacks — someone intercepting your WiFi traffic gets flagged instantly.

  • 😈

    TOR ANONYMITY + LOCATION PICKER

    Routes your traffic through Tor. Pick which country you appear to be from — US, Germany, Netherlands, Japan and 16 more. One or multiple at once.

  • 🔍

    PORT SCANNER

    Scans your phone and local network for exposed ports that attackers could use to get in.

  • ⏰

    AUTO-SCAN (SILENT MODE)

    Runs a full security scan every 2 minutes in the background. Starts automatically when your phone boots.

// How to install

Up and running
in 3 steps.

Works on any Android phone. No root. No special knowledge.

01

Install Termux from F-Droid

Download F-Droid from f-droid.org, then install Termux from there. Do not use the Play Store version — it's outdated.

02

Run the install command

Open Termux and paste this one command. It installs everything automatically.

03

Launch Ghost Mode

Type bash ~/ghost.sh — your security menu appears. You're protected.

curl -sL https://raw.githubusercontent.com/YOURUSERNAME/ghost-mode/main/setup.sh | bash

No SD card? No problem. Ghost Mode stores everything inside Termux's own storage — no external card needed.
